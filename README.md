# DonCEy_Kong_Jr
Proyecto en C y Java sobre el clasico juego de arcade Donkey Kong JR

Para correr en MacOs:
1. compilar: make clean                                
make
2. dar permisos de ser necesario: chmod +x create-terminal-launchers.sh
3. Crear ejecutables ./create-terminal-launchers.sh
4. Iniciar server
5. Iniciar instancias cliente

Para correr en Windows:
1. compilar: mingw32-make clean                                
             mingw32-make
2. Crear ejecutables: create-windows-launchers.bat
3. Iniciar server
4. Iniciar instancias cliente
